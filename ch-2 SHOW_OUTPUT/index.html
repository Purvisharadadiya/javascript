  • what is javascript

    JavaScript is a programming language used to make websites interactive. It runs in web browsers and helps with tasks like showing animations, updating web pages without reloading, and handling user actions like clicks or typing.

  • JavaScript is a free, open-source scripting language used for web development.

  • JavaScript is OOP(Object Oriented Programming) based Client Side Language.
  • JavaScript is Loosely Coupled Language and Weakly Typed Language and Case Sensitive Language.
  • JavaScript is not a compiled language, but it is a translated language. The JavaScript Translator (embedded in the browser) is responsible for translating the JavaScript code for the web browser.
  • JavaScript has no connectivity with Java programming language.
  • In addition to web browser, databases such as CouchDB and MongoDB uses JavaScript as their scripting and query language.

  • History of JavaScript

    1. Creation (1995):

    JavaScript was created by Brendan Eich in just 10 days while he was working at Netscape Communications Corporation. Initially named Mocha, it was later renamed to LiveScript and finally to JavaScript to align with the popularity of Java at the time, even though the two languages are not related.

  • 2. Purpose:

    JavaScript was designed to make web pages dynamic and interactive, allowing browsers to execute small scripts directly on web pages.

  • 3. Standardization (1997):

    JavaScript was submitted to ECMA (European Computer Manufacturers Association) International, which led to the creation of the ECMAScript standard. The first official version, ECMAScript 1, was released to ensure consistency across different browsers.

  • Es5 & Es6 concept Differences

    ES5

    It was introduced in 2009.

  • It has a lower performance as compared to ES6. Uses var to declare variables. Functions are written using the function keyword. String concatenation is done using +. Classes are created using constructor functions and prototypes. Default values are manually set.

  • ES6

    It was introduced in 2015.

    It has a higher performance than ES5. Introduced let and const for block-scoped variables. Introduced arrow functions for shorter syntax and automatic this binding. Introduced template literals using backticks `` for easier string interpolation. Introduced the class syntax for a cleaner and more readable approach. Default parameters are supported natively.
